【詳細政策建議】
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────

1. 立即行動方案（24小時內）
   • 檢討 345 個高風險避難所結構安全
   • 建立緊急轉移預案，影響 209,601 人
   • 通知相關單位準備應急物資與人員
   • 啟動高風險區24小時監測系統

2. 短期改善方案（1週內）
   • 評估 312 個中風險避難所加固需求
   • 規劃替代避難所位置，避開河川500m範圍
   • 建立分級預警發布機制
   • 更新應急撤離路線圖

3. 中期規劃方案（1個月內）
   • 進行詳細的地質與水文評估
   • 建立跨部門協調機制
   • 設置臨時避難所設備儲備點
   • 辦理避難所管理人員培訓

4. 長期發展方案（1年內）
   • 定期更新河川圖資與避難所資料（每季）
   • 結合氣象預報進行動態風險評估
   • 考慮氣候變遷長期影響評估
   • 建立智慧化避難所管理系統

【技術規格與限制】
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
• 資料來源：消防署避難收容所資料（967筆）、水利署河川圖資（13,262筆）
• 座標系統：WGS84 (EPSG:4326) → TWD97 (EPSG:3824) → WGS84
• 分析工具：Python 3.10 + GeoPandas + Shapely + Matplotlib + Pandas
• 緩衝區計算：歐幾里得距離，適合台灣地區
• 分析限制：基於靜態河川資料，未考慮即時水情
• 精度範圍：±50m（GPS定位精度）

【風險評估標準】
────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────
• 高風險（500m內）：需要立即檢討防護措施，建議轉移或加強
• 中風險（500-1000m）：需要密切監控，定期檢查設施狀況
• 低風險（1000-2000m）：相對安全，但仍需定期檢查
• 安全區（2000m外）：一般風險，正常維護即可
